# 🚛 新北市垃圾車時間查詢｜垃圾車幾點到？即時倒數＋日曆提醒

選你家附近的清運點，即時倒數垃圾車抵達時間，一鍵把每週收運時間加進手機日曆。純靜態網站，跑在 GitHub Pages 上，資料由 GitHub Actions 每月自動更新。

## ✨ 功能

- **下一班倒數**：以台北時間計算下一次收運（垃圾／回收／廚餘分開計），一小時內顯示分秒倒數
- **三種找清運點方式**：區→里瀏覽、關鍵字搜尋（路名／地標）、📍 GPS 定位找最近 20 個清運點（含距離）
- **每週時刻表**：一眼看懂週幾收垃圾 🗑️、回收 ♻️、廚餘 🥬
- **📅 加入日曆**：產生 .ics 檔（每週重複事件＋提前 10 分鐘提醒），iPhone／Android／Google 日曆都能匯入
- **⭐ 常用清運點**：收藏多個清運點（例如家裡＋公司），一鍵切換
- **🔗 可分享網址**：選好的清運點會寫進網址 hash，貼給家人直接開
- **記住上次選擇**：下次打開直接顯示你的清運點

## 🆚 為什麼不用官方網站就好？

我們實測了官方「[垃圾清運資訊查詢網](https://crd-rubbish.epd.ntpc.gov.tw/)」，痛點如下（本站逐一解決）：

| 官網痛點 | 本站做法 |
| --- | --- |
| 進站先彈全螢幕公告視窗，每次都要手動關 | 公告改為一行提示＋連結，不擋操作 |
| 以地圖為核心，地圖載入慢（實測 4 秒仍空白）、手機吃行動數據 | 純文字清單為主，全站不到 100KB（不含資料），地圖需要才開 Google Maps |
| 查詢入口藏在角落小按鈕，動線不直覺 | 打開就是選單，三步內看到結果 |
| 查詢邏輯是「選星期＋時段」，使用者要自己換算「下一班是什麼時候」 | 直接告訴你「下一班還有多久」，倒數計時 |
| 沒有提醒功能，要提醒得另外裝 App | 一鍵匯出 .ics，用手機內建日曆就能每週提醒 |
| 查詢結果沒有固定網址，無法分享 | 清運點寫進 URL hash，可直接分享連結 |
| 「之前查過」只存查詢條件，不能收藏清運點 | ⭐ 常用清運點收藏，家裡＋公司一鍵切換 |

## 🏗️ 架構

```
index.html                      ← 網站本體（無外部套件、單檔）
data/index.json                 ← 行政區索引（Actions 產生）
data/<行政區>.json               ← 各區清運點資料（Actions 產生）
scripts/fetch-data.mjs          ← 抓資料腳本（Node 20+，無外部套件）
.github/workflows/update-data.yml ← 每月 1 號自動抓最新資料
```

前端不直接呼叫政府 API（該 API 不支援 CORS），改由 GitHub Actions 在伺服器端抓資料、切成每區 JSON commit 進 repo，網站讀同源檔案：快、穩、離線可快取，也不怕 API 掛掉。

## 🚀 部署

1. Fork 或推這個 repo 到 GitHub
2. Actions 頁籤 → 「Update garbage truck data」→ Run workflow（第一次手動跑，之後每月自動）
3. Settings → Pages → Deploy from a branch → `main` / root
4. 完成，網址在 `https://<帳號>.github.io/<repo名>/`

## 📅 資料來源與授權

- 資料：[新北市資料開放平臺「新北市垃圾車路線」](https://data.ntpc.gov.tw/datasets/edc3ad26-8ae7-4916-a00b-bc6048d19bf8)（新北市政府環境保護局，政府資料開放授權條款）
- 共 26,000+ 個清運點、29 個行政區，含表定時間與垃圾／回收／廚餘週別
- ⚠️ 表定時間僅供參考，實際以現場為準；颱風等天災停收請看[環保局公告](https://crd-rubbish.epd.ntpc.gov.tw/)
